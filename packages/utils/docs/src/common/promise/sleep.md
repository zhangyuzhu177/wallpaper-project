# sleep

## 方法说明

使用 Promise 封装 setTimeout，使其支持 async/await

## 参数

| 参数名 | 类型 | 描述 | 默认值 |
| --- | --- | --- | --- |
| time | `number` | 等待的时长，单位为 ms |  |

## 返回值

| 类型 | 描述 |
| --- | --- |
| `Promise<unknown>` | 封装后的Promise |
