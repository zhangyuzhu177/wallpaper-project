# validatePassword

## 方法说明

校验一个密码是否符合要求

## 参数

| 参数名 | 类型 | 描述 | 默认值 |
| --- | --- | --- | --- |
| password | `string \| undefined` | 待校验的密码 |  |
| account | `string \| undefined` | 用户名 |  |

## 返回值

| 类型 | 描述 |
| --- | --- |
| `string` | 如果符合要求，返回空字符串，否则返回错误信息 |
