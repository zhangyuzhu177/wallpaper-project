# validateIp

## 方法说明

校验一个 ip 地址范围数组是否符合要求

## 参数

| 参数名 | 类型 | 描述 | 默认值 |
| --- | --- | --- | --- |
| arr | `string[] \| string[][] \| undefined` | 待校验的 ip 地址范围数组 |  |

## 返回值

| 类型 | 描述 |
| --- | --- |
| `string` | 如果符合要求，返回空字符串，否则返回错误信息 |
