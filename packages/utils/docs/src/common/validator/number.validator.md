# validateNumber

## 方法说明

校验一个数字是否符合要求

## 参数

| 参数名 | 类型 | 描述 | 默认值 |
| --- | --- | --- | --- |
| label | `string` | 参数标签 |  |
| value | `number \| undefined` | 待校验的数字 |  |
| min | `number \| undefined` | 最小值 |  |
| max | `number \| undefined` | 最大值 |  |

## 返回值

| 类型 | 描述 |
| --- | --- |
| `string` | 如果符合要求，返回空字符串，否则返回错误信息 |
