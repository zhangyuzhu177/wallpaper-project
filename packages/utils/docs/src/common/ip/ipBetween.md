# ipBetween

## 方法说明

判断 ip 地址是否在指定的数组范围内

## 参数

| 参数名 | 类型 | 描述 | 默认值 |
| --- | --- | --- | --- |
| ip | `string \| undefined` | ip 地址 |  |
| arr | `string[] \| string[][] \| undefined` | 指定的数组范围 |  |

## 返回值

| 类型 | 描述 |
| --- | --- |
| `boolean` | 是否在指定的数组范围内 |
