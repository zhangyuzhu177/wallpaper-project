# objectToArray

## 方法说明

将对象转换为对象数组

## 参数

| 参数名 | 类型 | 描述 | 默认值 |
| --- | --- | --- | --- |
| obj | `Record<K, V>` | 原始对象 |  |
| name1 | `N1` | 键对应的key |  |
| name2 | `N2` | 值对应的key |  |

## 返回值

| 类型 | 描述 |
| --- | --- |
| `(Record<N1, K> & Record<N2, V>)[]` | 转换后的对象数组 |
